Dog Engineering Calculator

💡 Executive Summary:
The Dog Engineering Calculator is an interactive, Tamagotchi-style learning tool where a pixel-art puppy reacts to your math results and evolves as you solve quizzes. It combines professional engineering calculation features with gamified education to make mathematics more engaging for students of all levels.

🐾 v1.0 Legacy Features:
The core features established in the initial version include:
 * Reactive Character: Pixel-art animations where the dog's expressions and dialogue change based on calculation results.
 * 3 Difficulty Levels: Specialized modes tailored for Middle School (Basic), High School (Trig/Log), and University (Advanced) math.
 * Basic Quiz Mode: A gamified learning experience featuring a 60-second timer per problem.
 * Scoring System: Awards 1.0 point for manual entry and 0.5 points for using complex calculator functions.
 * Multilingual Support: Full localized support for Korean, English, Japanese, and Chinese.

🦴 v2.0 New Features:
Version 2.0 introduces advanced growth mechanics and smart study tools:
 * Tamagotchi Evolution System: Accumulate EXP from correct quiz answers to evolve your pet through four stages: Egg → Puppy → Adult → Champion.
 * Smart Wrong Answer Notes: The system automatically records incorrect answers and provides specialized solution guides and hints provided by the dog character.
 * Archiving & Memos: Users can write custom notes on incorrect answers and move mastered items to a dedicated archive tab for better organization.
 * Scratch Pad Calculator: Access a secondary "Scratch" calculator during quizzes to perform intermediate calculations and copy results directly to the answer field.
 * Hearts & Streaks: Manage a life system (3 hearts) that decreases with mistakes, while gaining bonus EXP for maintaining correct answer "Streaks".
 * Enhanced Visuals: Includes evolution toast notifications and real-time "Method Badges" to show if you are currently earning full or half points.

🚀 How to Run:
This project is built using pure web technologies and requires no installation:
 * Download the Dog_Engineering_Calculator_v2.html file.
 * Open the file directly in any modern web browser (Chrome, Edge, Safari, etc.).
