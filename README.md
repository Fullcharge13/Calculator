# 🐶 멍멍이 공학 계산기 v2 (Dog Engineering Calculator v2)

🇰🇷 한국어 (Korean)
멍멍이 공학 계산기 v2는 귀여운 강아지 캐릭터와 함께 수학을 배우는 다마고치형 인터랙티브 학습 도구입니다. 사용자의 계산 결과와 퀴즈 정답 여부에 따라 강아지가 성장하고 반응합니다.

🐾 v1.0 기존 기능
강아지 캐릭터 반응: 계산 결과에 따라 강아지의 표정과 대사가 실시간으로 변화하는 픽셀 아트 애니메이션입니다.

3단계 난이도: 중학생(기본), 고등학생(삼각함수/로그), 대학생(고등 수학) 모드를 지원합니다.

기본 퀴즈 모드: 60초의 제한 시간 내에 문제를 푸는 게임화된 모드입니다.

차등 점수제: 수동 입력 시 1점, 계산기 함수 사용 시 0.5점을 획득합니다.

다국어 지원: 한국어, 영어, 일본어, 중국어를 지원합니다.

🦴 v2.0 새로운 기능
다마고치 진화 시스템: 퀴즈 정답을 맞혀 EXP를 쌓으면 알 → 강아지 → 성견 → 명견 순으로 진화합니다.

스마트 오답노트: 틀린 문제를 자동 저장하고, 강아지가 제공하는 풀이 가이드와 힌트를 확인할 수 있습니다.

아카이브 및 메모: 오답에 개인 메모를 남기거나 복습이 완료된 문제를 보관함(Archive)으로 옮길 수 있습니다.

스크래치 보조 계산기: 퀴즈 도중 별도의 계산기를 열어 연산하고 결과값을 답안으로 즉시 복사할 수 있습니다.

생명력 & 스트릭: 오답 3회 시 하트가 감소하며, 연속 정답(Streak) 시 보너스 EXP를 획득합니다.

🇺🇸 English
Dog Engineering Calculator v2 is an interactive Tamagotchi-style learning tool designed to grow alongside the user's math skills. Your pup reacts and evolves based on your calculation and quiz results.

🐾 v1.0 Existing Features
Reactive Character: Pixel art animations where the dog's expressions and dialogue change based on results.

3 Difficulty Levels: Specialized modes for Middle School (Basic), High School (Trig/Log), and University (Advanced).

Basic Quiz Mode: A gamified learning experience with a 60-second timer per problem.

Scoring System: Awards 1 point for manual entry and 0.5 points for using calculator functions.

Multilingual Support: Fully supports Korean, English, Japanese, and Chinese.

🦴 v2.0 New Features
Tamagotchi Evolution: Earn EXP from correct answers to evolve your pup from an Egg → Puppy → Adult → Champion.

Smart Wrong Answer Notes: Automatically records mistakes and provides specialized solution guides and hints.

Archiving & Memos: Write custom notes on incorrect answers and move mastered items to the archive.

Scratch Pad Calculator: Access a secondary helper calculator during quizzes and copy results directly to the answer field.

Hearts & Streaks: Manage life (hearts) lost from mistakes while gaining bonus EXP for correct answer streaks.
