## 🇰🇷 한국어

**냥이 공학 계산기**는 귀여운 고양이 캐릭터와 함께하는 인터랙티브 공학용 계산기 웹 애플리케이션입니다. 학생들의 수준에 맞춘 레벨별 계산 기능과 수학 실력을 테스트할 수 있는 퀴즈 모드를 제공합니다.

### ✨ 주요 기능
* **3가지 난이도 레벨**: 중학생(기본), 고등학생(삼각함수/로그), 대학생(역삼각함수/미적분 등) 모드 지원.
* **🐱 인터랙티브 냥이 UI**: 계산 결과에 따라 고양이의 표정과 대사가 변하며(Pixel Art 애니메이션 적용).
* **🎮 퀴즈 모드**: 60초 제한 시간 내에 문제를 푸는 게임화된 모드. 계산기 사용 여부에 따라 점수가 다르게 산정됩니다 (수동 입력: 1점, 계산기 사용: 0.5점).
* **🌐 다국어 지원**: 한국어, 영어, 일본어, 중국어 완벽 지원.
* **📜 계산 기록**: 이전 계산 및 퀴즈 기록을 추적하고 재사용 가능.

### 🚀 실행 방법
이 프로젝트는 순수 HTML/CSS/JS로 작성되었습니다. 추가적인 설치 없이 브라우저에서 바로 실행할 수 있습니다.
1. 이 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 더블 클릭하여 웹 브라우저에서 엽니다.

## 🇺🇸 English

**Cat Engineering Calculator** is an interactive engineering calculator web application featuring a cute pixel-art cat. It provides level-based calculation features tailored to students' grades and a Quiz Mode to test your math skills.

### ✨ Features
* **3 Difficulty Levels**: Middle School (Basic), High School (Trig/Log), and University (Inverse Trig/Calculus) modes.
* **🐱 Interactive Cat UI**: The cat's expressions and dialogue change based on your calculation results.
* **🎮 Quiz Mode**: A gamified mode where you solve problems within a 60-second time limit. Scoring varies based on whether you use special calculator functions (Manual: 1pt, Calc used: 0.5pt).
* **🌐 Multilingual Support**: Fully supports Korean, English, Japanese, and Chinese.
* **📜 History**: Tracks your calculation and quiz history for easy reuse.

### 🚀 How to Run
This project is built with pure HTML/CSS/JS. You can run it directly in your browser without any installation.
1. Clone or download this repository.
2. Double-click the `index.html` file to open it in your web browser.

## 🇯🇵 日本語

**にゃんこ工学計算機**は、かわいいドット絵の猫キャラクターと一緒に使えるインタラクティブな工学用計算機ウェブアプリです。学生のレベルに合わせた計算機能と、数学のスキルを試すことができるクイズモードを提供します。

### ✨ 主な特徴
* **3つの難易度レベル**: 中学生（基本）、高校生（三角関数/対数）、大学生（逆三角関数/微積分など）モードをサポート。
* **🐱 インタラクティブな猫UI**: 計算結果に応じて猫の表情やセリフが変わります。
* **🎮 クイズモード**: 60秒の制限時間内に問題を解くゲームモード。特殊な関数を使用するかどうかでスコアが変わります（手動入力：1点、計算機使用：0.5点）。
* **🌐 多言語対応**: 韓国語、英語、日本語、中国語を完全サポート。
* **📜 計算履歴**: 過去の計算やクイズの履歴を保存し、再利用可能です。

### 🚀 実行方法
このプロジェクトは純粋なHTML/CSS/JSで作られています。追加のインストールは不要で、ブラウザですぐに実行できます。
1. このリポジトリをクローンまたはダウンロードします。
2. `index.html` ファイルをダブルクリックして、ウェブブラウザで開きます。

## 🇨🇳 中文

**猫咪工程计算器**是一款带有可爱像素猫角色的交互式工程计算器 Web 应用程序。它提供适合不同学生水平的分级计算功能，以及测试数学技能的测验模式。

### ✨ 主要功能
* **3种难度级别**：支持初中生（基础）、高中生（三角函数/对数）和大学生（反三角函数/微积分等）模式。
* **🐱 交互式猫咪 UI**：猫咪的表情和台词会根据计算结果而变化。
* **🎮 测验模式**：在60秒限时内解题的游​​戏化模式。分数会根据是否使用计算器特殊函数而有所不同（手动输入：1分，使用计算器：0.5分）。
* **🌐 多语言支持**：完美支持韩语、英语、日语和中文。
* **📜 历史记录**：跟踪您的计算和测验历史记录，方便再次使用。

### 🚀 运行方法
本项目使用纯 HTML/CSS/JS 编写。无需任何额外安装，直接在浏览器中运行即可。
1. 克隆或下载此仓库。
2. 双击 `index.html` 文件即可在 Web 浏览器中打开。
