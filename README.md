## 🇰🇷 한국어

**멍냥이 공학 계산기**는 귀여운 강아지와 고양이 캐릭터가 계산 결과에 반응하는 인터랙티브 웹 애플리케이션입니다. 학생들의 수준에 맞춘 레벨별 계산 기능과 수학 실력을 테스트할 수 있는 퀴즈 모드를 제공합니다.

### ✨ 주요 기능

* **🐶🐱 캐릭터 테마**: 계산 결과에 따라 강아지와 고양이의 표정 및 대사가 변하는 Pixel Art 애니메이션.
* **3가지 난이도 레벨**: 중학생(기본), 고등학생(삼각함수/로그), 대학생(역삼각함수/미적분 등) 모드 지원.
* **🎮 퀴즈 모드**: 60초 제한 시간 내에 문제를 푸는 게임화된 모드. 계산기 사용 여부에 따라 점수가 다르게 산정됩니다 (수동 입력: 1점, 계산기 사용: 0.5점).
* **🌐 다국어 지원**: 한국어, 영어, 일본어, 중국어 완벽 지원.
* **📜 기록 관리**: 이전 계산 및 퀴즈 기록을 추적하고 재사용 가능.

### 🚀 실행 방법

이 프로젝트는 순수 HTML/CSS/JS로 작성되었습니다. 추가적인 설치 없이 브라우저에서 바로 실행할 수 있습니다.

1. 이 저장소를 클론하거나 다운로드합니다.
2. `Cat Engineering Calculator__v1.html` 혹은 `Dog Engineering Calculator__v1.html` 파일을 웹 브라우저에서 열어 실행합니다.

## 🇺🇸 English

**Dog-Cat Engineering Calculator** is an interactive web application where cute dog and cat characters react to your math results. It provides level-based calculation features for students and a Quiz Mode to test your skills.

### ✨ Features

* **🐶🐱 Pet Themes**: Pixel Art animations where character expressions and dialogue change based on results.
* **3 Difficulty Levels**: Middle School (Basic), High School (Trig/Log), and University (Advanced) modes.
* **🎮 Quiz Mode**: A gamified mode with a 60-second time limit. Scoring varies: Manual input earns 1pt, while using calculator functions earns 0.5pt.
* **🌐 Multilingual Support**: Fully supports Korean, English, Japanese, and Chinese.
* **📜 History**: Tracks calculation and quiz history for easy reuse.

### 🚀 How to Run

Built with pure HTML/CSS/JS. Run directly in your browser without installation.

1. Clone or download this repository.
2. Open `Cat Engineering Calculator__v1.html` or `Dog Engineering Calculator__v1.html` in your web browser.
